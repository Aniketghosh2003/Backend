**ORM (Object-Relational Mapping):** Maps objects in code to relational database tables (SQL). Used in SQL databases like MySQL, PostgreSQL. Example: Sequelize (Node.js), Hibernate (Java).  
**ODM (Object-Document Mapping):** Maps objects in code to document-based databases (NoSQL). Used in NoSQL databases like MongoDB. Example: Mongoose (MongoDB).
